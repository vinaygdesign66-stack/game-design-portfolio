### Recommendation for your portfolio

Since you're creating this to apply for **Game Designer** roles, I'd refine **Breath of Shadows** into a **4-mission campaign** instead of 8. A concise, polished GDD is more likely to be read by hiring managers than an oversized document. Four highly detailed missions demonstrate pacing, systems, progression, and storytelling without making the project feel unrealistically large for a solo designer.

We can now build this document **chapter by chapter**, preserving a consistent style and ending with a complete professional GDD.
